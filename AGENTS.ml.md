# ML Research Workspaces

- On session start in a research workspace (a `competition/<name>/` folder, or any directory carrying an
  experiment `README.md`): **read the current directory's `README.md` first** — it is the workspace entry
  point and names the rest of the reading order. A bare "continue" from a workspace folder means exactly
  that: README → its listed docs → the frontier.
- **`sdlc` scope:** only use `sdlc` for full-blown software development projects. Research investigations
  run on the lighter ml-research workflow (the ledger + verification commits, single-threaded on `main`),
  not the SDLC spine.
- **Communication:** use less jargon — explain things to a smart but out-of-domain operator.
